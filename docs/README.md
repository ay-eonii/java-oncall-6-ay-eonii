## 구현 기능 목록

### 월과 시작 요일 정보를 입력

- [x] 월과 시작 요일을 입력받는다.
    - 월은 1월부터 12월까지 입력 가능하다.
    - 시작 요일은 [일, 월, 화, 수, 목, 금, 토] 입력 가능하다.
- [x] 평일 비상 근무 순서를 입력받는다.
- [x] 휴일(토요일, 일요일, 법정공휴일) 비상 근무 순서를 입력받는다.

### 비상 근무일 배정

- [ ] 비상 근무자는 어떤 경우에도 연속 2일은 근무할 수 없다.
    - [ ] 순번상 특정 근무자가 연속 2일 근무하게 되는 상황에는, 다음 근무자와 순서를 바꿔 편성한다.

### 비상 근무표 출력

- [x] 비상 근무표를 출력한다.
    - `X월 X일 (요일) (이름)`
